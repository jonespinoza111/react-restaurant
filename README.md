# react-restaurant

A website for a Chinese restaurant called the Hot Star Hunan Grill. It has a landing page, a full menu, and allows you to place orders.
Features and technology included in this project are:

- Landing Page
- Menu
- Online Ordering
- Cart System
- MERN stack
- Stripe Payments
